# SharpXDecrypt

## 声明

本程序仅供个人恢复密码使用，如果代码中存在侵权行为，请联系我们删除。



## 简介

Xshell全版本凭证一键恢复工具，针对Xshell全版本在本地保存的密码进行解密，包括最新的7系列版本！
